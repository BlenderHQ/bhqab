Log Messages (bhq_addon_base.utils_progress)

The module provides the ability to display the progressbar (and even several progressbars) in the status bar of the Blender. This technique can be used mainly with modal operators that run for a relatively long time and require the output of the progress of their work.

class bhq_addon_base.utils_progress.progress

A class that implements the initialization and completion of progressbars

class ProgressPropertyItem

Progress bar item that allows you to dynamically change some display parameters.
